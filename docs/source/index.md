# Welcome to Lumache's documentation!

**Lumache—Ü** (/lu'make/) is a Python library for cooks and food lovers
that creates recipes mixing random ingredients.
It pulls data from the [Open Food Facts database](https://world.openfoodfacts.org/)
and offers a *simple* and *intuitive* API.

Check out the {doc}`usage1` section for further information, including
how to {ref}`installation1` the project.

:::{note}
This project is under active development.
:::
